Order example

Below is an example about the power of the order. (I didn't mention it that way...)

  • The boxes are "sorted" by featured order, easily with .featured { order: -1; }
  • The buttons are sorted differently in each box, aligned with auto margin
  • Even the main parts can be moved, reordered ( box 4).
